# Ghost Messenger

**Ghost Messenger** — это мессенджер, созданный для тех, для кого **анонимность** — не просто слово.

Мы используем **сквозное шифрование (E2EE)**:  
все сообщения шифруются на устройстве отправителя и расшифровываются только на устройстве получателя.  
Ключи хранятся **только у вас и вашего собеседника** — ни сервер, ни мы, ни кто-либо другой не может их получить.

### Основные особенности (на текущий момент)

**E2EE** (Xchacha)
- Сообщения хранятся локально (Room) — не пропадают при перезапуске
- Анти-скриншот (FLAG_SECURE) — можно включить в настройках
- Скрытие статуса "онлайн" и "печатает..." (Stealth Mode)
- Кастомные темы, аватарки, обои чата
- Уведомления с предпросмотром (FCM)
- Panic Button — стереть все данные одним нажатием

### Текущий статус проекта

**v0.6.7 Beta** — активно разрабатывается  
Работает 1-на-1 чат, шифрование, базовые настройки.  
Но это **бета** — возможны баги, недоделки и изменения API.

### Что пока НЕ работает / в планах

- Групповые чаты  
- Реакции, упоминания, пересылка  
- Таймер самоуничтожения сообщений  
- Полный оффлайн-режим  
- Верификация ключей (safety numbers / QR)  
- Голосовые / видео сообщения
